### Beschreibung
Dies ist eine einfache **To-Do / Einkaufsliste** Webanwendung.  
Benutzer können Elemente zur Liste hinzufügen, Elemente löschen und ihre Einkäufe oder täglichen Aufgaben verwalten.

### Funktionen
- Neue Elemente zur Liste hinzufügen.
- Elemente mit der roten "X"-Taste löschen.
- Stilvolles Design mit Hover-Effekten.
- Responsives Layout für verschiedene Bildschirmgrößen.

### Verwendete Technologien
- HTML
- CSS
- JavaScript

### Verwendung
1. Öffnen Sie die Datei `index.html` in Ihrem Browser.
2. Geben Sie den Produkt- oder Aufgabennamen in das Eingabefeld ein.
3. Klicken Sie auf die **"Hinzufügen"**-Schaltfläche oder drücken Sie **Enter** (falls implementiert), um das Element zur Liste hinzuzufügen.
4. Klicken Sie auf die **rote X-Taste**, um ein Element aus der Liste zu entfernen.

# To-Do List / Einkaufsliste

## English Version

## Live Demo https://gogunskiy99.github.io/to_do_list

### Description
This is a simple **To-Do / Shopping List** web application.  
Users can add items to the list, delete items, and manage their shopping or daily tasks.

### Features
- Add new items to the list.
- Delete items with a red "X" button.
- Stylish design with hover effects.
- Responsive layout for different screen sizes.

### Technologies Used
- HTML
- CSS
- JavaScript

### How to Use
1. Open `index.html` in your browser.
2. Type the product or task name in the input field.
3. Click the **"Add"** button or press **Enter** (if implemented) to add it to the list.
4. Click the **red X** button to remove an item from the list.
